# User Guide

## üöÄ Getting Started

### Registration
1.  Navigate to the **Sign Up** page.
2.  Enter your **Institutional Email** (must end in `@1.ustp.edu.ph`).
3.  Select your **Course/Program**.
4.  Create a secure password.
5.  *Note: Only valid institutional emails are accepted to ensure community security.*

### Login
1.  Use your registered email and password using the **Login** page.
2.  Upon successful login, you will be redirected to the **Home Feed**.

## üìù Features & How-to

### Creating a Post
1.  Click the **"Create Post"** button (usually a `+` icon or designated button).
2.  Select the **Category**:
    *   **Lost & Found**: Available to ALL users.
    *   **Announcement/Event**: Available ONLY to **Organization Heads** and **Moderators**.
3.  Fill in the details (Title, Description, Images).
4.  Submit the post.

### Messaging (Lost & Found)
1.  Find a Lost & Found post you have information about.
2.  Click the **Message** button on the post card.
3.  This opens a direct message thread with the post owner linked to that specific item.

### Search
*   Use the search bar at the top of the feed.
*   Enter keywords to match post **Titles**.
*   Search is context-aware (Global or specific to the current section).

### Witness Feature
*   If you can vouch for a Lost & Found item (e.g., you saw it where the poster claims), you can mark yourself as a **Witness**.
*   This adds credibility to the post.

### Saved Posts
*   Click the **Bookmark/Save** icon on any post to add it to your **Saved Posts** list for quick reference later.
